# **App Name**: Profolio

## Core Features:

- Profile Showcase: Display your professional profile with a summary, photo, and social links.
- Project Gallery: Showcase your key projects with descriptions, images, and links to live deployments or code repositories.
- Skills Visualization: Visually represent your skills and expertise, such as with a bar graph or word cloud.
- AI-Powered Resume Generator: Tool for dynamically creating customized resume content tailored to different job applications.
- Contact Form: Allow visitors to send you messages directly through the website.

## Style Guidelines:

- Dark background (#121212) for a modern and sophisticated look.
- Primary color: A muted, elegant purple (#B19CD9). The selection of purple takes inspiration from a modern style combined with an essence of creativity.
- Accent color: A contrasting desaturated yellow (#E6D761).
- Body font: 'Inter', a sans-serif font providing a clean and modern feel, well-suited for body text.
- Headline font: 'Space Grotesk', a sans-serif font to be used in conjunction with the Inter font family to create unique headings.
- Use a set of minimalist icons for navigation and representing skills.
- Ensure a responsive layout that adapts to different screen sizes.
- Subtle animations for page transitions and interactive elements.
